## ATAC-seq Analysis

## Overview

This directory contains the computational workflows used to analyze chromatin accessibility changes in human neutrophils following short-term antibiotic treatment.

ATAC-seq was used to identify differentially accessible chromatin regions (DARs) associated with microbiota disruption and to link regulatory changes with transcriptional responses observed in RNA-seq.

The analysis integrates:
- chromatin accessibility profiling
- differential accessibility testing
- peak annotation
- transcription factor motif enrichment
- functional enrichment analysis
- correlation with RNA-seq differential expression

## Data Processing Pipeline

### ATAC-seq Processing

Raw sequencing reads were processed using the nf-core/atacseq pipeline implemented in Nextflow.

The pipeline performs:
- read quality control
- adapter trimming
- alignment to the human genome
- duplicate removal
- peak calling
- consensus peak generation
- signal track generation
- QC reporting

Key parameters include alignment to the GRCh38 genome, Bowtie2 alignment, and MACS2 peak calling.

The pipeline was executed using Nextflow with Docker containers.

The pipeline uses a reproducible software environment defined in ```software_versions.yml```, including tools such as Bowtie2, MACS2, SAMtools, BEDTools, deepTools, and Picard.

See ```run_nextflow.sh``` for execution command and ```params.yaml``` for pipeline parameters.

### Downstream Analysis (R)

Downstream analyses were performed in R using count matrices generated from consensus peaks.

The workflow includes:
- removal of unwanted variation
- differential accessibility analysis
- peak annotation
- functional enrichment analysis
- transcription factor motif enrichment
- correlation with RNA-seq data

The full analysis workflow is available in scripts/* and notebooks/ATACseq_analysis.Rmd

## Reproducibility

The computational environment can be reproduced using ```sessionInfo()``` included at the end of the scripts.

## Data availability

Raw sequencing data are available in NCBI BioProject: PRJNA1402444

Intermediate objects are not included in the repository but can be generated by running the analysis scripts.
